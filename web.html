Grundlæggende web

Responsiv stilartside

I denne opgave skulle vi researche forskellige design stilarter i grupper.
Derefter skulle vi lave en responsiv hjemmeside med HTML og CSS samt efterligne det grafiske udtryk for den givne stilart. Vi fik en wireframe vi skulle følge, for at lære hvordan det er bedst at sætte tingene op i html og lærte at lave et html skeleton.
Vi lærte også at bruge flexbox, ændre farver, sætte billeder og skriftyper ind og hvordan man tilpasser dem i størrelsen.

Dekonstruktivisme

Vores gruppe fik stilarten "dekonstruktivisme" fælles lave et moodboard og en præsentations pitch.
Forneden kan du klikke videre min stilartside eller den pitch min gruppe lavede.

Brugte værktøjer

  • Brackets (codeeditor) hermed HTML og CSS
  • Adobe Photoshop
  • Adobe Illustrator
