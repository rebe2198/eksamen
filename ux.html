Grundlæggende UX

Responsiv stilartside

I denne opgave skulle vi researche forskellige design stilarter i grupper.
Derefter skulle vi lave en responsiv hjemmeside med HTML og CSS samt efterligne det grafiske udtryk for den givne stilart.

Dekonstruktivisme

Vores gruppe fik stilarten "dekonstruktivisme" fælles lave lave moodboard og en præsentation.

Brugte værktøjer

  • Brackets (codeeditor) hermed HTML og CSS
  • Adobe Photoshop
  • Adobe Illustrator
